\# CodeScene Review



Analiza el código actual siguiendo principios de CodeScene.



\## Checklist:

1\. Identifica archivos modificados en staging/últimos commits

2\. Evalúa complejidad ciclomática

3\. Detecta funciones >50 líneas

4\. Busca duplicación de código

5\. Revisa acoplamiento entre módulos

6\. Genera reporte con prioridades



Output: Tabla markdown con hallazgos y prioridades.
